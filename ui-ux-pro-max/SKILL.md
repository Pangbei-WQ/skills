# UI/UX Pro Max Skill

This skill provides "Design Intelligence" by mapping specific product domains to their optimal visual DNA. It leverages a structured knowledge base of styles, colors, and UX guidelines.

## Capabilities
1. **Domain Detection**: Automatically identifies the product niche (e.g., AI Agent, Fintech, SaaS).
2. **Visual DNA Mapping**: Retrieves primary colors, typography, and animation styles for the specific domain.
3. **UX Auditing**: Provides 99+ targeted UX guidelines with severity levels.
4. **Platform Optimization**: Tailors designs for Mobile, Web, and Desktop.

## Usage
- Before designing any UI, invoke this skill to get the "Design Reasoning".
- Providing the domain name (e.g., "Web3 Dashboard") to get specific styling tokens.
- Use the provided `core.py` and `design_system.py` to automate token generation.

## Knowledge Base
- **Reasoning**: `data/ui-reasoning.csv`
- **Styles**: `data/styles.csv`
- **Colors**: `data/colors.csv`
- **UX Guidelines**: `data/ux-guidelines.csv`
