# Using Superpowers

## Overview
Superpowers is a collection of 14 high-level engineering skills designed for professional software engineering.

## The 14 Core Skills
1. `using-superpowers` (Coordination)
2. `writing-plans` (Planning)
3. `executing-plans` (Execution)
4. `systematic-debugging` (Investigation)
5. `test-driven-development` (Quality)
6. `subagent-driven-development` (SDO Roles)
7. `dispatching-parallel-agents` (Efficiency)
8. `using-git-worktrees` (Isolation)
9. `writing-skills` (Evolution)
10. `brainstorming` (Design)
11. `requesting-code-review` (Audit)
12. `receiving-code-review` (Feedback)
13. `verification-before-completion` (Gate)
14. `finishing-a-development-branch` (Finalization)

## Core Principle
Follow the process, even when it feels slower. The process prevents thrashing and ensures first-time success.
