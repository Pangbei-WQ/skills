# Writing Plans

## Core Principle
Planning is cheap, implementation is expensive. Resolve all ambiguity in the plan.

## Plan Structure
1. **Implementation Plan**: Numbered list of verifiable, independent tasks (5-15 mins each).
2. **Walkthrough**: Narrative explanation of the target code structure and flow.

## Workflow
1. Draft plan -> 2. Human Review -> 3. Lock plan.
Once locked, do not change significantly without discussion.
