# Writing Skills

## Goal
To extend the assistant's capabilities with specialized knowledge or workflows following TDD principles.

## Process
1. **Define Intent**: What is the skill for?
2. **Write SKILL.md**: Use YAML frontmatter and clear markdown instructions.
3. **Add Resources**: Files, templates, or helper scripts in the skill folder.
4. **Test & Verify**: Ensure the skill correctly guides the agent in a test scenario.
